## 关键学习

### 2026-03-19: Skill 驱动的优势

1. **无需代码实现**
   - 传统方式需要写状态机、流程引擎
   - Skill 方式：AI 直接理解执行

2. **易于维护**
   - 修改流程只需更新 Markdown
   - 不需要修改代码和重新部署

3. **AI 友好**
   - Markdown 是 AI 训练数据的常见格式
   - AI 能理解标题、列表、表格、代码块

4. **人类友好**
   - 任何人都能编辑 Markdown
   - 无需编程知识
